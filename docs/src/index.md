# julia\_mzML\_imzML.jl

*Julia support for loading mzML/imzML files*

## Package Features
- Load mzML files
- Load imzML files
- Retrive mz Slices from imzML files
- Save slices as bitmaps with zero memory and TrIQ quantization methods
